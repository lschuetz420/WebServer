### WEBSERVER

## Setup tutorial:

# 1. Download repo files and extract "target" folder
# 2. Download MySQL
# 3. Open MySQL CommandLine and paste the Database creation script -> enter
# 4. Open Terminal and navigate to "target" folder
# 5. Type "java -jar .\target\WebServer-1.0.jar start" -> enter
# 6. To link server to an address type "java -jar .\target\WebServer-1.0.jar startDefineProperties"
